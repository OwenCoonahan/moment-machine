# Moment Machine — Project Context

## What This Is
AI-powered content generation platform for SMBs during live events (Super Bowl).

## Live URL
https://moment-machine.vercel.app

## Repo
https://github.com/OwenCoonahan/moment-machine

## Current Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Deployed on Vercel

## API Keys Available
- `FAL_KEY` — Fal.ai for image/video generation
- `GROQ_API_KEY` — Groq for fast LLM
- `GOOGLE_AI_API_KEY` — Gemini
- `HEYGEN_API_KEY` — HeyGen for avatar videos

## ESPN API
Base: `https://site.api.espn.com/apis/site/v2/sports/football/nfl/`
- `/scoreboard` — Live scores
- `/summary?event={id}` — Game details
